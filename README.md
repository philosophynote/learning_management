# learning_management
- 学習時間・内容を記録するサイトです。
- 学習した時間・内容を登録・参照することができるとともに棒グラフなどで可視化できます。
- ストップウォッチやタイマーも搭載しているので、このサイト内で学習時間の記録は完成します。

# 使用技術
- HTML5
- CSS3
- Bootstrap5
- Javascript（plotly.js）
- PHP 7.4.12
- MySQL 5.7

#　機能一覧
- ユーザ登録・ログイン機能
- ストップウォッチ
- タイマー
- 簡易ToDoリスト
- 学習時間記録・表示機能
  - グラフによる可視化
